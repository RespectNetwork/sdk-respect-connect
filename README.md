Respect Network SDK for Respect Connect
=======================================

This is an SDK for Respect Connect requesting parties.

### How to build

First, you need to build the main [XDI2](http://github.com/projectdanube/xdi2) project.

After that, just run

    mvn clean install

To build all components.
